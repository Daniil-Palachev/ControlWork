Создаем первый массив (array1), в котором будут содержаться слова с различным количеством символов.
Создаем метод (ThreeSymbols), в котором инициализируем второй массив (array2), далее будем проходиться по всем объектам первого массива на проверку необходимых условий и в случае выполнения условия, слово из первого массива будет перемещаться во второй.
По результатам метода (ThreeSymbols) выводим содержимое второго массива, в котором будут слова из первого массива с количеством символов <=3.
